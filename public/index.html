Pagaska API meneruskan request kamu ke host utama secara otomatis — cukup kirim endpoint, sisanya kami yang urus.

Operasional
Base URL
https://your-project.vercel.app/api
Cara Penggunaan
Contoh Request
# Format URL GET /api{endpoint}?{query} # Contoh — endpoint AI Allen GET /api/ai/allen?text=halo # curl curl "https://your-project.vercel.app/api/ai/allen?text=halo"
Catatan: API key disisipkan otomatis oleh server — kamu tidak perlu menyertakannya di request. Parameter apikey dari client akan diabaikan demi keamanan.
